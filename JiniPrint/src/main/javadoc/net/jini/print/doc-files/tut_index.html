Introduction

The following Lessons introduce you to the Jini Print Service. They are written from the client's perspective and show you how to write code to use the Jini Print Service to make things print out. General familiarity with Jini is assumed.

Lesson 1. The Very Basics. This lesson shows you how to print out that most proverbial of messages, "Hello, world!"

Lesson 2. Print Data Attributes. This lesson shows you how to tell the Print Service how you want it to print your job.

Lesson 3. Lookup. This lesson shows you how to find a Print Service that can do what you want in the first place.

Lesson 4. Print Job Status. This lesson shows you how to monitor the status of your print job.

Lesson 5. Print Job Events. This lesson shows you how to track the progress of your print job by receiving event notifications.

Lesson 6. File Print Jobs. This lesson shows you how to print a file stored locally on the client.

Lesson 7. Internet File Print Jobs. This lesson shows you how to print a document stored at a URL outside the client.

Lesson 8. Service Registration. This lesson shows how a Jini Print Service instance would register itself into the Jini Lookup Service (JLUS).
